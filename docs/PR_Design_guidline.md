| 名前         | カラーコード | 用途                                     |
| ------------ | ------------ | ---------------------------------------- |
| Primary Navy | `#1A237E`    | タイトル・セクション番号背景・アクセント |

## Figma制作フロー

### ステップ1：フレーム設定

```
スライドサイズ：1920 × 1080px（16:9）
または：1280 × 720px（軽量版）
```

### ステップ2：コンポーネント作成（先に作ると効率的）

1. セクション番号ボックス（Navy正方形）
2. カードコンポーネント（白背景・角丸・ドロップシャドウ）
3. 年代タグ（角丸・Navy背景・白文字）
4. キーワードボックス（Light Blue背景）
5. 棒グラフ要素

### ステップ3：スライド制作順序

1. タイトルスライド（デザインの基準を作る）
2. カラー/フォントStyleの登録
3. 各スライドをコンポーネントを組み合わせて作成

### ステップ4：Figma Slides機能の活用

- **プレゼンテーションモード：** Figmaで直接発表可能
- **アニメーション：** Smart Animate で要素のフェードイン設定
- **プロトタイプ：** スライド間をArrowキーで操作するように接続

---

## 注意事項・Tips

- **Auto Layout** を積極活用 → 要素の間隔・余白が自動調整
- **Styles登録** → Color StylesとText Stylesを最初に登録すると一括変更が楽
- **コンポーネント化** → 繰り返し使う要素（カード等）は必ずコンポーネントに
- **グリッドガイド：** 各フレームに`96px左右余白`のGrid Layoutを設定
